Intro

Installable via npm install --save datastore-s3, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save datastore-s3

You can then require() datastore-s3 as normal:

const datastoreS3 = require('datastore-s3')

In the Browser

datastore-s3 should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of datastore-s3, including index.js and index.min.js, are included in the /dist folder. datastore-s3 can also be found on unpkg.com under

  • https://unpkg.com/datastore-s3/dist/index.min.js
  • https://unpkg.com/datastore-s3/dist/index.js
